## 🚀 Як використовувати

### **1. Відкрийте оновлену сторінку:**
```
http://localhost:3000/ai-matching
```

### **2. Побачите список ваших активних вакансій**

### **3. Використовуйте кнопки:**
- **"Генерувати для всіх"** - знаходить кандидатів для всіх вакансій одночасно

- ✅ **Швидко** - результат за 2-3 секунди
